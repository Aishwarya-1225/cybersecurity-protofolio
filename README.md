What I Learned

Information Security vs Cybersecurity

*Information Security protects all types of information, whether digital or physical.
*Cybersecurity focuses on protecting computers, networks, systems, and digital data from cyber threats.

CIA Triad

*Confidentiality: Ensures information is accessible only to authorized users.
*Integrity: Ensures data remains accurate and unaltered.
*Availability: Ensures information and systems are available when needed.

Defense in Depth

*A security strategy that uses multiple layers of protection to reduce the risk of attacks.

Zero Trust Architecture

*Based on the principle: "Never Trust, Always Verify."
*Every user and device must be verified before accessing resources.

Security Controls

*Measures implemented to prevent, detect, and respond to security threats.

GRC
*Governance: Defines policies and responsibilities.
*Risk Management: Identifies and manages security risks.
*Compliance: Ensures adherence to laws, regulations, and standards.

Due Care vs Due Diligence

*Due Care: Taking appropriate actions to protect assets.
*Due Diligence: Identifying and assessing risks before implementing security measures.

Threat Actors
*Individuals or groups that attempt to compromise systems or data for various motives.

Open Source Intelligence (OSINT)
*Collecting information from publicly available sources to support investigations and security analysis.

Indicators of Compromise (IoC

*Unusual outbound network traffic

 Malware
What is Malware?
*Malware is malicious software intentionally designed to damage systems or steal data.
*It can:
*Steal personal information
*Delete files
*Perform click fraud
*Steal software license keys
First Malware
Creeper (created in the early 1970s)
How Malware Infects Systems
1.Attacker sends a malicious link or attachment.
2.Victim clicks the infected file or link.
3.Malware installs and begins stealing, modifying, or destroying data.
Types of Malware
*Worm
*Virus
*Trojan
*Logic Bomb
*Spyware
*Adware
*Ransomware
*Botnet
 Ransomware
Learned:
*Encrypts files and demands payment for recovery.
*Can lock users out of their systems.
Common Infection Methods:
*Malicious email attachments
*Phishing links
*Instant messages
Prevention:
*Keep software updated
*Enable firewalls
*Avoid suspicious emails and links

 Viruses
Learned:
*A virus attaches itself to executable files.
*Executes when the infected program runs.
*Can spread through USB drives, CDs/DVDs, and other removable media.

Types of Viruses:
*Master Boot Record (MBR) Virus
*File Infector Virus
*Macro Virus
*Script Infection Virus
Advanced Virus Techniques:
*Multipartite Virus
*Stealth Virus
*Polymorphic Virus

 Encryption & Hashing
Encryption
*Converts Plaintext → Ciphertext using an encryption key.
*Used to protect confidential information.

Hashing
*Converts Plaintext → Hash Value using a hash function.
*One-way process mainly used for integrity verification and password storage.
